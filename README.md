# InsightAgent

AI-powered Polymarket intelligence agent — Agora Hackathon (Canteen x Circle x Arc)

## What it does

InsightAgent monitors Polymarket prediction markets in real-time, analyzes news sentiment, and recommends +EV bets using Kelly Criterion sizing — settled on Arc Testnet with Circle Developer-Controlled Wallets.

## Features

- Real-time market data from Polymarket API
- News sentiment analysis via NewsAPI
- Kelly Criterion bet sizing
- Circle Developer-Controlled Wallets on Arc Testnet
- Web dashboard with live market signals

## Stack

- Python + Flask
- Polymarket Gamma API
- NewsAPI
- Circle Wallets SDK
- AWS EC2 Ubuntu 24.04
